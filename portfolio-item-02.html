Raise The Bar and Wellness Spa App

Built with Swift in Xcode

Overview: This project is a native iOS Gym & Spa app developed to showcase real-world SwiftUI development and API integration skills. The app features a full membership experience, including booking, shopping, and profile management.
  • 📱 Multi-tab structure: Home, Booking, Store, Workout, Profile
  • 🔐 OAuth2 authentication with Mindbody Partner API (demo limited)
  • 🛍️ Store with mock inventory and cart functionality
  • 📆 Booking flow for gym classes and spa sessions
  • 💾 Persistent login via Keychain, profile image storage

Tech Stack: SwiftUI, Combine, OAuth2, Keychain, MVVM Architecture

Note: This app is currently in active development. Demo access is unavailable at this time, but the screenshots below reflect an earlier working build and core features.
